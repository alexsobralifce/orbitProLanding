# Orbit Pro - Soluções em IA
==========================

Bem-vindo ao repositório oficial da **Orbit Pro**. Nossa landing page apresenta nossos principais serviços em consultoria, agentes inteligentes, Data Science e treinamentos corporativos focados em Inteligência Artificial.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando tecnologias modernas visando alta performance e facilidade de manutenção:

*   **Framework:** [Next.js 15 (App Router)](https://nextjs.org/)
*   **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
*   **Estilização:** [Tailwind CSS v4](https://tailwindcss.com/)
*   **Animações:** [Framer Motion](https://www.framer.com/motion/) e Canvas API nativa
*   **Formulários:** [React Hook Form](https://react-hook-form.com/)
*   **Ícones:** [Lucide React](https://lucide.dev/)

## 🛠️ Como Executar o Projeto Localmente

Siga as instruções abaixo para rodar o projeto em seu ambiente local:

### 1. Requisitos
Certifique-se de ter o [Node.js](https://nodejs.org/) instalado na sua máquina.

### 2. Instalação

Clone o repositório ou entre na pasta do projeto e instale as dependências:

```bash
npm install
```

### 3. Rodando o Servidor de Desenvolvimento

Para iniciar o servidor Next.js localmente, execute:

```bash
npm run dev
```

Abra o seu navegador e acesse [http://localhost:3000](http://localhost:3000) para ver a página.

## 📂 Estrutura do Projeto

A estrutura principal das pastas do projeto está organizada de forma a facilitar o rápido desenvolvimento de novos componentes de interface:

```text
├── src/
│   ├── app/                # Rotas, Root Layout e Styles globais (Next App Router)
│   └── components/         # Componentes React independentes da Landing Page
│       ├── Navbar.tsx      # Barra de navegação responsiva
│       ├── Hero.tsx        # Seção principal com chamada e partículas animadas
│       ├── Services.tsx    # Nossos principais pilares de atuação
│       ├── Differentials.tsx
│       ├── Contact.tsx     # Formulário de Lead + Contatos Diretos
│       └── NeuralNetwork.tsx # Animação exclusiva de fundo
```
